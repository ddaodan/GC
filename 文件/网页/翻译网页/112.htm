P 1-1-2 月球
玩家到达的第一个星球，是地球的卫星，表面有很多陨石造成的环形山

月球重力仅为主世界的18%，重力会影响到玩家、怪物、箭的飞行轨迹与丢出物品的飞行轨迹
从16格以上的地方掉落玩家才会受到伤害，127格高度摔落才会致命

日夜循环
月球的日夜循环需要 2h40min，相当于主世界8天，月球昼夜也与主世界显示的月相有关。白天在月球上将会持续1h20min，相当于主世界4天，从主世界看，这将会是满月到下一个新月。晚上将会是从新月到下一个满月。下图为月相图，上半部分为当月球早上时主世界看月球是什么样的，下半部分为当月球晚上时主世界看月球是什么样的



天空
在月球上看主世界，你会感觉主世界不再动，只有太阳在动，但其实这并不完全正确，主世界也是在动的（地球绕太阳转）因为系统设定玩家在月球赤道之上，所以看起来主世界是静止的

玩家在月球上向南移动时，主世界看起来是在向北移动，玩家向北移动时，主世界看起来是在向南移动。如果玩家坐标在z=11500或z=-7000时，主世界看起来将会在月球地平线。如果玩家坐标在z=2250，那么在玩家看来，地球与太阳将会在特定时间点上形成一条直线，这便是在月亮上看日蚀，不过日蚀发生时，月球亮度并不会降低，很遗憾

时间指令
月球日出 = 主世界第一个日出 = /time set 0000 = 现实时间: 00h00min
月球中午 = 主世界第三个日出 = /time set 48000 = 现实时间: 00h40min
月球日落 = 主世界第五个日出 = /time set 96000= 现实时间: 01h20min
月球午夜 = 主世界第七个日出 = /time set 144000 = 现实时间: 02h00min
月球日出 = 主世界第九个如初 = /time set 192000 = 现实时间: 02h40min

地质特征
月球表面覆盖着地球草皮，向下一层覆盖着月球泥土，再向下一层覆盖着月岩，再向下能够找到月球铜矿、月球锡矿以及奶酪矿石。月球并不会自然生成岩浆
有时你能够在月球表面找到陨石（挖掘后掉落陨铁）
